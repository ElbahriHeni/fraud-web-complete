# Fraud Web Complete

A complete React + TypeScript + Vite starter for the fraud management web frontend.

## Included screens
- Login
- Forgot Password
- Dashboard
- Fraud Queue
- Case Details / Case Management
- Reports List
- Report Details
- User Management List
- Add User / User Details

## Run in VS Code
1. Extract the zip.
2. Open the folder in VS Code.
3. Open Terminal.
4. Run:
   ```bash
   npm install
   npm run dev
   ```
5. Open the local URL shown in the terminal, usually `http://localhost:5173`.

## Notes
This project uses mock data only. No backend is connected yet.
